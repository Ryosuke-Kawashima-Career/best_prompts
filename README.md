# Best Prompts

A personal collection of prompts gathered through practice. These prompts are organized by use case and refined over time to get the most effective results from AI assistants.

## Structure

```text
best_prompts/
├── agent-settings/          # Prompts and configs for AI agent behavior
│   ├── commands-workflows/  # Slash commands and workflow prompts
│   ├── rules/               # Heuristics and behavioral rules
│   └── skills/              # Reusable skill definitions
├── daily-tasks/             # Prompts for everyday tasks
│   ├── Email/               # Email drafting and refinement
│   ├── LearningTutor/       # Learning and tutoring sessions
│   ├── Lecture/             # Lecture Q&A and takeaways
│   ├── LinkedIn/            # LinkedIn post improvement
│   └── LogicalThinking/     # Logical question generation
├── docs/                    # Documentation and walkthroughs
└── specs/                   # Requirements and planning specs
```
